# 📖 QuranSphere

Live Demo: https://quran-sphere-9mmv.vercel.app/


A modern, responsive, and production-ready Quran web application built with React, TypeScript, Tailwind CSS, and shadcn/ui. QuranSphere provides a clean and distraction-free reading experience with translations, tafsir, audio recitation, and powerful search capabilities.

---

## ✨ Features

- 📖 Complete Quran Reader
- 📚 Surah, Juz & Hizb Navigation
- 🌍 Multiple Translations (English & Bangla)
- 📜 Tafsir Support
- 🎧 Audio Recitation
- 🔍 Powerful Search (Arabic, English & Bangla)
- 🌙 Dark & Light Theme
- 🔤 Adjustable Font Size & Reading Settings
- 📍 Continue Reading
- 📑 Last Read Position
- 📤 Copy & Share Ayah
- 📱 Fully Responsive Design
- ⚡ Fast Performance
- ♿ Accessibility (WCAG)
- 🔎 SEO Optimized
