# Project: NoorQuran
Mission: Modern, fast, minimalist Quran app.
 
# Tech Stack
- Next.js (App Router, TypeScript, Tailwind CSS)
- LocalStorage for user preferences
- API: fawazahmed0/quran-api
 
# Rules for Claude
- Always prioritize performance and "Zero-Cost" architecture.
- Keep components modular. Use @/components, @/hooks, @/lib paths.
- No database needed; use React Context/Zustand for client-side state.
- Focus on clean, readable code and Tailwind utility classes.
- Use responsive design (mobile-first).
- Before implementing a new feature, always propose a brief plan/architecture.
